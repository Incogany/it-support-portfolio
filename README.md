- Operating systems
- Networking
- Security basics
- Troubleshooting
- Documentation
- Help desk workflow
- Scripting

## Repository Structure

- **docs** — setup notes, standards, and supporting documentation
- **labs** — hands-on practice in OS, networking, and security
- **projects** — support-focused portfolio projects
- **troubleshooting** — issue-resolution guides
- **tickets** — realistic help desk scenarios
- **scripts** — support and automation scripts
- **kb** — knowledge base articles
- **resume** — professional background material

To build a professional IT support portfolio that demonstrates technical ability, troubleshooting logic, and clear documentation.
